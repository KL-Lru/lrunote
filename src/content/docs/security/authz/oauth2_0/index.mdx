OAuth 2.0 は自身のサービスが保有するリソースに対して, サードパーティアプリケーションがアクセスすることを認可するためのフレームワークの 1 つです.

用途やセキュリティ要件に応じて様々なフローが定義されており, 適切なフローを選択することで, ユーザのリソースを安全に保護しつつ, アプリケーション間の連携などを実現させることができます.
